Flex box CSS

CSS Flexible Box Layout, commonly known as Flexbox, is a CSS 3 web layout model. It is in the W3C's candidate recommendation stage.

Alignment and space

The flex layout allows responsive elements within a container to be automatically arranged depending upon screen size.

tricky positioning

The flex directional flow can be specified as rightwards, leftwards, upwards, or downwards.
